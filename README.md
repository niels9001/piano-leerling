# 🎹 Piano Leerling

Een web app om keyboard te leren spelen! Speciaal gemaakt voor kinderen.

## Features

- 🎵 **Noten Leren** — Leer de toetsen kennen en noten lezen op de notenbalk
- 🎶 **Liedjes Spelen** — Speel bekende kinderliedjes stap voor stap
- 🎹 **Vrij Spelen** — Oefen vrij op het virtuele keyboard

## Hoe te gebruiken

1. Open de app op je iPad (of andere browser)
2. Kies wat je wilt doen: noten leren, liedjes spelen of vrij spelen
3. Bij **Noten Leren**: werk door de 4 niveaus heen
4. Bij **Liedjes Spelen**: tik op de gemarkeerde toetsen om het liedje te spelen
5. Je voortgang wordt automatisch opgeslagen!

## Technologie

- HTML5 / CSS3 / JavaScript
- [Tone.js](https://tonejs.github.io/) voor geluid
- [VexFlow](https://www.vexflow.com/) voor muzieknotatie
